# Ali Nikovic Finance Portfolio

A simple, professional portfolio site tailored to finance, with an AI-powered stock research assistant.

## Run Locally

1. Make sure Node.js 18 or newer is installed.
2. Copy `.env.example` to `.env`.
3. Add your Anthropic API key:

```txt
ANTHROPIC_API_KEY=your_api_key_here
```

4. Start the site:

```bash
npm start
```

Then open `http://localhost:3000`.

## Important

The stock analyzer is for research and education only. It should not be treated as financial advice.
